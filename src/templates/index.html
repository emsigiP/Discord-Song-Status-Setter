How do I get my token?

Method 1: Using Developer Tools

  1. Open Discord in your web browser (Chrome, Edge, Firefox, etc.) and log in.
  2. Press F12 (or Ctrl+Shift+I) to open Developer Tools.
  3. Go to the Network tab at the top.
  4. Type /api in the filter/search box.
  5. Click on any server or channel in Discord to load some activity.
  6. Click on any network request that appears in the list (like science, profile, or settings).
  7. Look at the Headers tab on the right side, scroll down to Request Headers.
  8. Find the authorization header and copy the long string of text next to it.
